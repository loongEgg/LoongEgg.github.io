GithubPage-1.极简github入门和静态网页生成

0. 摘要

  • 主题

    在github上创建一个仓库，并学会使用git和VSCode发布一个简单的GithubPage网页

  • 步骤

    1. 创建github仓库
    2. 在本地电脑初始化git并配置
    3. 发布到Github Page

  • 注意

    • 代码和操作中的[]中的内容需要自己适应性修改
    • 本次示范不适合不使用[YOUR_GITHUB_NAME].github.io命名的代码仓库

1. 创建github仓库

  • 在github网页右上角 + > New Reposity
  • 创建一个名为[YOUR_GITHUB_NAME].github.io的代码仓库（PS: 这样的话生成github page后的访问域名为https://[YOUR_GITHUB_NAME].github.io，看起来像自己的官方网站）
  • 直接Creat Reposity不需要设置readme之类的
  • 复制一下代码仓库地址，比如我的是https://github.com/innergeek/innergeek.github.io.git

2. 在本地初始化Git并配置

  • 在电脑合适的地方创建一个文件夹，比如我的为E:\blog\innergeek.github.io
  • 初始化Git
    • 在文件夹中 右键 > Git bash here
    • 在命令行中输入git init
    • 关联到远端代码仓库在命令行中输入git remote add origin https://github.com/[YOUR_GITHUB_NAME]/[YOUR_GITHUB_NAME].github.io.git， 我的示范命令为`git remote add origin https://github.com/innergeek/innergeek.github.io.git
